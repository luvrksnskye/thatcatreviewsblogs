The story behind this tiny website!

Skye Journey Indie Web & Game Dev

Hi everyone! I'm Skye, the developer and designer behind this website. I wanted to take a moment to share how I built this blog system and why I decided to do it this way. Plus, I thought it would be a nice way to leave my client with a template for all her future posts.

The Idea

Everything started when Abby asked me for a blog that was easy to use. She wanted something simple... and well, huh, "simple" and I don't always get along.

My solution was straightforward: just create an HTML file and let the system handle everything else. No databases, no admin panels, no headaches.

How the Magic Works

The system is made of three main parts:

  • The smart server - Keeps an eye on the blog folder and detects when you add or update posts
  • The manifest generator - Reads each post's meta tags and builds a JSON file with all the info
  • The dynamic Status Tab - Automatically shows the latest posts on the homepage

The Details

I had a lot of fun adding small touches that make the site feel special:

  • The floating clouds in the background
  • The softly falling stars
  • The calming pastel colors
  • The smooth little animations everywhere

For the Tech-Curious

The tech stack is intentionally simple:

  • HTML/CSS/JavaScript - Pure vanilla, no frameworks
  • Node.js - Only for the dev server
  • File System Watcher - To detect real-time changes
  • CSS Variables - For the light/dark theme system
  • ES6 Modules - To keep the code clean and organized

No React, no Vue, no complicated build steps. Just clean code that anyone can read and tweak.

For now, the system works exactly the way Abby needs it to - and that's what matters most.
